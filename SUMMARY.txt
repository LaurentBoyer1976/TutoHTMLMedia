## 🎨 STYLES CSS CLÉS

```css
/* Container principal */
.definition-section {
  padding: 2rem;  /* ← Padding comme demandé */
  background: gradient;
  border-top: 3px solid #667eea;
}

/* Layout côte-à-côte */
.definition-container {
  display: flex;
  gap: 2rem;  /* Espace entre article et cartes */
}

/* Article à gauche */
.definition-article {
  flex: 1;  /* Prend l'espace disponible */
  padding: 2rem;
  border-left: 4px solid #667eea;
}

/* Cartes à droite */
.definition-sidebar {
  flex: 0 0 320px;  /* Width fixe */
  display: flex;
  flex-direction: column;
  gap: 1.5rem;  /* Entre les cartes */
}

/* Responsive */
@media (max-width: 1024px) {
  .definition-sidebar {
    flex-direction: row;  /* Cartes en ligne */
  }
}

@media (max-width: 768px) {
  .definition-container {
    flex-direction: column;  /* Full-width */
  }
}
```
